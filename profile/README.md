# GuildMasterDev

A guild of open-source resource hubs — each one a specialist in its craft. Every app is free, works offline as a PWA, and has a live demo.

## Built With

Every guild member runs as an Electron desktop app and deploys as a static web demo or PWA. No accounts, no tracking, no ads. Built with vanilla JS, Angular, React, Svelte, and TypeScript — whatever fits the craft.
